##  Project Overview

*This module is based on a **Coursera Guided Project**, focusing on the practical application of audience segmentation techniques.*

In this project, you will learn how to use segments inside Google Analytics and how they provide a deeper analysis of your website visitors. You will learn how to set up and use segments. You will learn about system segments and custom segments. You will also learn how to import segments. You will understand how using segments helps you make better decisions when it comes to increasing and converting your website traffic.

##  Learning Objectives

By the end of this analysis, the following competencies are demonstrated:

- [x] **LO1:** Understand segments and the benefits of using segments.
- [x] **LO2:** Understand and use system segments.
- [x] **LO3:** Create custom segments.
- [x] **LO4:** Use advanced options for custom segments.
- [x] **LO5:** Import Segments.
- [x] **LO6:** Managing Segments.

###  Data Science Application
*From a Data Science perspective, this project serves as the data collection and feature engineering layer. The segments identified here (System & Custom) are exported to Python to train the **K-Means Clustering** models found in this repository.*
